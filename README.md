# 📊 EnerPred – Energy Prediction Web App

EnerPred adalah aplikasi berbasis web yang digunakan untuk memprediksi energi (khususnya energi surya) berdasarkan parameter lingkungan yang dimasukkan oleh pengguna.

Aplikasi ini dibangun menggunakan Streamlit sehingga dapat digunakan langsung melalui browser tanpa instalasi tambahan.

---

## 🌐 Akses Aplikasi

Gunakan aplikasi melalui link berikut:
https://enerpred.streamlit.app/

---

## 🚀 Cara Menggunakan

### 1. Buka Website
Buka link aplikasi menggunakan browser seperti Chrome, Edge, atau Firefox.

### 2. Masukkan Data Input
Isi parameter yang tersedia pada form, seperti:
- Intensitas cahaya matahari
- Suhu lingkungan
- Kelembaban
- Parameter lain sesuai yang tersedia di aplikasi

### 3. Klik Tombol Prediksi
Klik tombol **Predict / Hitung / Submit** untuk memproses data.

### 4. Lihat Hasil
Hasil prediksi akan ditampilkan secara langsung, berupa:
- Nilai prediksi energi
- Visualisasi grafik (jika tersedia)
- Informasi tambahan

---
